# MSG Конструктор турниров

Веб-приложение для организации спортивных турниров: от расчёта расписания до ввода счетов и печатного протокола. Работает как PWA — устанавливается на телефон, работает офлайн.

**Живой сайт:** https://saselle.github.io/msg-tournaments/

## Возможности

- **Параметры турнира** — команды (4–512), поля, дни, круговая/плей-офф/смешанная система
- **Автопланировщик** — расписание с учётом перерывов команд, заблокированных слотов, разных окон для разных дней
- **Ввод счетов в приложении** — тапом по матчу, крупные +/− кнопки
- **Живые турнирные таблицы** — очки, разница, места пересчитываются мгновенно
- **Автопродвижение по сетке плей-офф** — победители сразу подставляются в следующий раунд
- **Импорт команд списком** — вставляешь имена, получаешь готовый состав
- **Логотипы/цвета команд** — цветные полосы для быстрой идентификации
- **Судейский планшет** — упрощённый ввод счёта через QR-код
- **Excel-экспорт** — полный турнир с формулами
- **PDF-протокол** — итоговый документ в фирменном стиле MSG

## Установка на телефон

1. Открой сайт в Chrome / Safari
2. Меню → «Установить приложение» / «На экран Домой»
3. Появится иконка с трофеем, откроется без адресной строки

## Структура репозитория

```
├── index.html              # Готовый билд (то что открывается пользователем)
├── src/
│   ├── tournament_builder.jsx   # Основной React-компонент (главный исходник)
│   ├── qrcode.js                # Библиотека генерации QR-кодов
│   ├── xlsx_style.js            # SheetJS с поддержкой стилей

│   ├── tailwind.js              # Tailwind CDN
│   ├── icon.svg                 # Векторный источник иконки
│   ├── icon-192.png             # PWA иконка малая
│   └── icon-512.png             # PWA иконка большая
└── README.md
```

## Как собирать локально

Нужны Node.js 18+ и npm.

```bash
# Установить esbuild и React
npm install esbuild react react-dom qrcode-generator fflate

# Собрать бандл
npx esbuild src/tournament_builder.jsx \
  --bundle --minify --jsx=automatic \
  --loader:.jsx=jsx --external:xlsx \
  --define:process.env.NODE_ENV='"production"' \
  --outfile=app_bundle.js
```

Затем всё вложить в `index.html` — исходный шаблон сборки лежит в комментариях основного файла.

## Технологии

- React 18 + xlsx-js-style (Excel c формулами и стилями)
- Tailwind CSS (утилитарные классы через Play CDN)
- fflate (перепаковка xlsx для активации автоматических формул)
- qrcode-generator (SVG QR-коды)
- localStorage (хранение состояния турнира)
